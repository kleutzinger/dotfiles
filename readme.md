## Keybindings (OS)
All keybindings are defined in [config/sxhkd/sxhkdrc](.config/sxhkd/sxhkdrc)
* `F1`: dropdown terminal
* `super + enter`: floating terminal
* `super + space`: dmenu_run
* `super + w`: open browser
* `super + m`: maximize
* `super + q`: quit application
